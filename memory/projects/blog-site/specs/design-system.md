# Blog Site Design System

## Colors
- `--bg`: #07070f (background)
- `--surface`: #0d0d1a (cards)
- `--surface2`: #13132b (nested)
- `--border`: #1e1e3a
- `--purple`: #7c3aed
- `--purple-light`: #a855f7
- `--cyan`: #06b6d4
- `--text`: #e2e8f0
- `--text-muted`: #64748b
- `--green`: #4ade80

## Typography
- Headings/Body: 'Space Grotesk', sans-serif
- Mono: 'Space Mono', monospace

## Components
- Featured post card
- Post grid (2-col)
- Sidebar with profile/store/links
- Navbar (hide on scroll)
- CTA boxes (gradient purple→cyan)
- Social share buttons

## Layout
- Max width: 1100px
- Grid: 1fr 300px (main + sidebar)
- Mobile: 1fr, sidebar static
